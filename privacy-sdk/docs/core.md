````markdown
# Core SDK (`packages/core`)

This document explains the core SDK package and how to use the `ContractClient` locally.

Overview

- `packages/core` contains the TypeScript SDK used by dApps to interact with `TransactionRegistry` and verifier contracts. It includes helper utilities used by tests and examples.

Getting started

1. Install dependencies:

```cmd
cd packages\core
pnpm install
```

2. Run unit tests:

```cmd
cd packages\core
````
